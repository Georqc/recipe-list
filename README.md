# MovieSwipe

A React app that fetches movies from the TMDB API.  
Users can browse, swipe, and save movies they want to watch.

---

## Features

### Technical

- User authentication
- Fetch movies from the public TMDB API
- Swipe functionality
  - Swipe right → save the movie
  - Swipe left → skip
  - Both actions load a new movie

### Functionality

- Search for users
- Save movies to “My List”
- Rate saved or viewed movies
